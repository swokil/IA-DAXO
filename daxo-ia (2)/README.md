
# Daxo IA - Intelligence Supérieure

Ce projet est une plateforme d'IA multi-modale (Chat, Image, Vidéo, Voix).

## Installation Locale

1. Clonez le dépôt.
2. Installez les dépendances : `npm install`.
3. Créez un fichier `.env` à la racine et ajoutez :
   `API_KEY=VOTRE_CLE_GOOGLE_AI_STUDIO`
4. Lancez le projet : `npm run dev`.

## Déploiement

Connectez ce dépôt à **Vercel** ou **Netlify** et n'oubliez pas d'ajouter la variable d'environnement `API_KEY`.
